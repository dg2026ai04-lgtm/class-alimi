# 🌸 2학년 2반 온라인 알리미

수행평가, 시험 일정, 준비물, 공지사항을 한곳에서 확인하는 귀여운 Streamlit 앱입니다.

## 주요 기능
- 📌 이번 주 주요 일정
- 📝 수행평가 / 제출 기한
- 📚 시험 범위
- 🎒 준비물
- 💬 익명 건의함 (풍선 애니메이션 🎈)
- 📣 시험기간 응원 안내
- 🔒 관리자 모드 (탭으로 항목별 추가/삭제)

## 특징
- 파스텔 색감 디자인 🌸
- 풍선/눈 애니메이션 효과
- 모든 항목을 화면에서 추가/삭제 가능
- 처음엔 비어 있고, 관리자가 직접 채워넣는 방식

## 실행 방법
```bash
pip install -r requirements.txt
streamlit run app.py
